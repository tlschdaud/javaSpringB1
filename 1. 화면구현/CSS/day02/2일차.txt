CSS
1. 선택자
1) 태그 선택자 - 태그명
    - 복수개 선택
    - 제일 넓은 적용범위
    - * : 모든 태그 - 사이트에 적용되는 공통적인 속성(예 - 폰트 종류 ...)

2) 클래스 선택자(class 속성 - 띄어쓰기로 여러개 클래스 지정)
    - 복수개 선택 가능
    - .클래스명

3) 아이디 선택자 (id 속성)
    - 한개 선택
    - #아이디명

 태그  <  클래스(.)   <아이디(#)     <  스타일 속성(style)

!important; : 우선순위 강제 적용

4) 조상 자손 선택자

조상 선택자 .... 자손 선택자

5) 부모 자식 선택자

부모 선택자 > 자식 선택자

6) 그룹 선택자
선택자, 선택자, ....{ 공통 스타일 }

7) 가상 클래스 선택자
: 
- 상태를 선택
선택자:hover - 마우스가 올라간 상태
         : checked - 체크된 상태
         : selected - 선택된 상태
         : focus - 포커스가 된 상태
         
- 조건, 위치 선택
선택자:not(제외할 선택자) : 제외할 선택자를 제외하고 선택
         :first-child : 첫번째 자식 선택
         :last-child : 마지막 자식 선택
         :nth-child(수치) : 수치 번째 선택(1번 부터 시작)
                            (수식):
                             예) : nth-child(2n) - > 짝수 번째 선택
                                  : nth-child(2n+1)-> 홀수 번째 선택
                                    
         :first-of-type : 첫번째 요소 선택
         :last-of-type : 마지막 요소 선택
         :nth-of-type(수치 또는 수식)

- 가상의 요소 추가 선택
     :after - 자식의 마지막에 추가
     :before - 자식의 가장 첫번째에 추가

          - content 속성이 없으면 추가 X, 반드시 정의
             : 내용이 없어도 content:'' 정의 해야 한다.


2. 결합자
- 특정 기호 : 관계를 좀더 좁혀주는 기호
- 부모 > 자식
- 일반 형제 결합자
    선택자 ~ 형제 선택자 :  오른쪽 형제를 전체 선택

- 인접 형제 결합자
    선택자 + 인접 형제
      li+li = 첫번째 제외 나머지 선택


참고)
선택자.클래스명 -> 선택자에 클래스명이 있는 선택자
선택자#아이디명 -> 선택자에 이이디명이 있는 선택자

선택자[속성명="속성값"]



속성
1. 폰트
- font-size : 글자의 크기
    크기 단위
    
     고정 크기
         px : 픽셀  - 화소 기준 - 1화소 - 1px

     상대 크기 : 
         rem : 가장 상위 태그의 폰트 사이즈 1rem
           
         em : 부모 태그의 폰트 사이즈 1em

- color : 글자의 색상
    1) 색상명
         예) color: orange;
    2) rgb(0~255, 0~255, 0~255)
        rgba(...)
               - a : alpha 채널 - 투명도
                    0~1 : 0에 가까울 수록 투명, 1에 가까울 수록 불투명
     3) hex code(16진수)
          R                         G                       B
     # 00~ff(255)    0~ff(255)        0~ff(255)

     #ff   00   00 : 빨강 = #f00
        R    G    B

     #cccccc = ccc
     #ffffff = 흰색 = #fff
     #000000 = 검정색 = #000

참고) 반복되거나 예상 가능한 부분은 하나로 줄여서 가능
참고)
       투명도
                opaciity: 0~1; 0에 가까울 수록 투명, 1에 가까울 수록 불투명
    
   text-align : 좌, 우, 가운데
              left - 왼쪽 (기본값)
              right - 오른쪽
              center - 가운데
              justity - 양쪽 정렬

    line-height : 행간 - 행과 행 사이의 간격
        1.2 - 기본값 - 실제 높이 20% 만큼 간격 띄운다

        배율, px 고정 사이즈

    letter-spacing : 자간 - 글자 사이 간격
    word-spacing : 단어 사이 간격
    
    font-family : 글꼴을 설정
          font-family : 폰트명, 폰트명, 폰트명 ....;
 
          - 시스템 폰트 : 현재 PC, 현재 서버에 설치된 기본 폰트
          - Malgun Gothic

    font-weight : 글의 두께
           normal - 보통
           bold - 두껍게

           수치
           100~900 : 폰트마다 지원 범위가 다르다

2. 공간
   블록(Block) 태그
      - 줄개행
      - 너비, 높이 지정
      - 외부여백 - 상하좌우 적용 가능
         (margin)
      - 내부여백 - 상하좌우 적용 가능
          (padding)

   인라인(Inline)
      - 줄개행
      - 너비, 높이 지정 X
      - 외부 여백 - 좌우 여백 적용 가능
         (margin)
      - 내부여백 - 상하좌우 적용 가능
         (padding)

    display 속성
          block - 블록 속성으로 변경
          inline - 인라인 속성으로 변경
          inline-block; - 인라인 + 블록(너비, 높이, 외부 여백)
          
          none - 안보임 처리

                 참고)
                      - visible - 보임
                      - hidden - 안보임

          flex - 배치
